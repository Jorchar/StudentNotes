# StudentNotes
IT Studies - engineering thesis defence

PROJECT AND IMPLEMENTATION OF A MOBILE APPLICATION FOR CREATING NOTES ON ANDROID SYSTEM

Application uses Firebase realtime database and authentication.
MLKit was used to implement OCR module.

Required Android 8.0 or higher
